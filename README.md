## 🐍 **Introduction**
Python is one of the most popular programming languages due to its simplicity and readability.  
This repository provides hands‑on examples to help you understand the basics of Python programming through small, easy‑to‑follow scripts.

---

## ✔️ **Prerequisites**
Before you begin, you should have:
- Basic computer knowledge  
- Interest in learning programming  
- A laptop/PC with internet access  

No prior coding experience is required.

---

## 💻 **Installing Python**

### **Windows / macOS / Linux**
1. Visit the official Python website: https://www.python.org/downloads  
2. Download the latest stable version (Python 3.x).  
3. Install Python and ensure you check the box:  
   **“Add Python to PATH”** (Windows only).  
4. Verify installation:  
   ```bash
   python --version
   ```

---

## ▶️ **Running Your First Python Program**

### **Option 1: Using a Code Editor (Recommended)**
Install any editor:
- VS Code  
- PyCharm  
- Sublime Text  

Create a file named `hello.py`:
```python
print("Hello, Python!")
```

Run it:
```bash
python hello.py
```

### **Option 2: Using Python Shell**
Open terminal and type:
```bash
python
```
Then:
```python
print("Hello World")
```
